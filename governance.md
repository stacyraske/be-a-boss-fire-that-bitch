# Governance Protocol

## Ownership
- **Primary Author**: Stacy Raske
- **Publisher**: Empowered Overthinker Publishing
- **Repository Maintainer**: Stacy Raske (or designated archive manager)

## Versioning Rules
- All updates follow semantic versioning (`MAJOR.MINOR.PATCH`)
- `CHANGELOG.md` must be updated before any merge
- Manuscript text is archival and immutable post-publication
- Frameworks and wiki pages may be updated as methodology evolves

## Update Workflow
1. Draft changes in a feature branch
2. Verify against published manuscript for accuracy
3. Update `CHANGELOG.md` with summary and date
4. Pull request → author review → merge to main

## Access Control
- `CODEOWNERS` routes approvals to Stacy Raske or designated publisher representative
- External contributions require explicit written permission
