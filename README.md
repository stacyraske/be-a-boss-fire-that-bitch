## Overview
This repository archives the complete manuscript of *Be a Boss & Fire That Bitch* alongside an extracted IP/framework layer. The dual-layer structure preserves the raw archival text while making signature frameworks, diagnostic language, and developmental models machine-readable and permanently referenceable.

## Directory Structure
```
/
├── README.md
├── CHANGELOG.md
├── canonical_citation.md
├── LICENSE.md
├── manuscript/          # Raw archival text (chapter-by-chapter)
├── front-matter/        # Foreword, TOC
├── back-matter/         # Acknowledgments, About the Author
├── frameworks/          # Extracted IP/models (machine-readable)
└── wiki/                # Navigation layer (mapped to repo files)
```

## Usage
- **Manuscript**: Reference for original text, citations, and editorial tracking.
- **Frameworks**: Standalone markdown files containing extracted models, processes, and diagnostic language.
- **Wiki**: Human-readable navigation, glossary, and structural mapping.
